# 项目整理记录

当前仓库主要包含 2025 平安城市相关样例图片和两份可见算法脚本：

| 文件 | 作用 |
| --- | --- |
| `人偶分类/标记职业人数.py` | 通过 SIFT + FLANN 做模板匹配，统计并标记职业人员 |
| `红绿灯/traffic lights recognition.py` | 通过 HSV 颜色分割和位置规则识别红、黄、绿灯 |

`垃圾桶/` 和 `楼宇/` 目前更像数据目录，建议后续补充算法说明或任务说明，避免读者不知道这些图片应该如何使用。
