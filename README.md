# 2025 RAICOM 平安城市

本项目整理 2025 睿抗机器人开发者大赛平安城市赛题相关资料与算法代码，主要面向第一次参加 RAICOM 平安城市赛题的同学。仓库保留了不同视觉任务的样例数据、初版算法和处理结果，方便同学提前了解任务形式、输入输出要求和常见解题思路。

平安城市赛题通常围绕机器人视觉感知展开，需要参赛者根据图像或视频信息识别道路环境、人员目标、建筑状态和交通设施。相比单纯调用模型，比赛更看重对场景特征的理解、图像预处理、规则判断和结果表达。

## 团队分工

本项目为团队协作成果，具体开发分工如下：
- **人偶分类**：由本人负责核心算法设计、代码编写与调试。
- **其他任务（红绿灯、垃圾桶、楼宇等）**：由团队其他队员协同开发、数据整理与代码维护。

## 任务说明

### 人偶分类

`人偶分类/标记职业人数.py` 使用 SIFT 提取模板图像和目标图像的特征点，再通过 FLANN 匹配器寻找相似区域。程序会根据单应性矩阵估计模板在目标图中的位置，并通过角度、面积和凸四边形等条件过滤误匹配。

运行后，结果会保存到 `人偶分类/match_results/`。代码中的 `len(valid_matches)` 对应当前图像中识别到的职业人员数量。

### 红绿灯识别

`红绿灯/traffic lights recognition.py` 将图像转换到 HSV 颜色空间，分别提取红色、黄色、绿色区域。程序会根据灯的排列方向、亮灯位置和颜色面积判断当前信号灯状态，并把“红灯”“黄灯”“绿灯”等结果写回图片。

运行后，结果会保存到 `红绿灯/hld/processed_results/`。

### 垃圾桶与楼宇

`垃圾桶/` 和 `楼宇/` 目前主要保存样例图片，可作为后续目标检测、颜色识别、OCR 或规则判断算法的输入数据。建议后续把对应算法单独整理为 `.py` 文件，并在文档中补充运行命令和输出格式。
